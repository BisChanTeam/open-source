# ⚙️ Установка

### 1. Клонируйте репозиторий

```bash
git clone <ссылка-на-репозиторий>
```

или скачайте ZIP-архив.

---

### 2. Перейдите в папку проекта

```bash
cd open-source
```

---

### 3. Установите зависимости

```bash
npm install
```

---

### 4. Запустите сервер

```bash
npm start
```

После запуска сайт будет доступен по адресу:

```
http://localhost:3000
```
